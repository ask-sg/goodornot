# 作品好不好

这是按《书法作品评价体系 v1.2》的制作的移动端优先静态网站原型，适合先用于微信传播演示。

## 已实现

- 作品流浏览、搜索、筛选、排序
- 刷新或再次打开后，未评价作品优先展示，并轮换不同作品到前面
- 增加 Wikimedia Commons 公开图源测试作品
- 单作品支持最多4张图片，作品卡片下方提供缩略图切换
- 每件作品支持单独转发，链接打开后优先展示该作品
- 未评价前隐藏作者与评价数据
- 演示版账号注册、登录、退出
- 超级管理员账号和管理平台
- “好 / 孬”一次性评价
- 投票后展示好孬标签、好评值、好评率、评价人数、作者信息
- 评价者权重、阶段奖励、单次评价历史
- 作品发布表单与图片预览
- 个人主页、评价历史、个人作品数据
- 微信分享标题、描述和封面图基础信息

## 本地预览

当前目录已启动本地预览服务：

```text
http://localhost:4173/
```

也可以直接双击打开：

```text
index.html
```

## 微信传播

把本目录中的 `index.html`、`styles.css`、`app.js` 和 `assets/` 上传到支持 HTTPS 的静态网站服务后，就可以把链接发到微信。当前账号、评价、留言等数据保存在访问者自己的浏览器里；正式上线前建议接入真实后端，用服务器保存用户、作品、评价记录和权重结算数据。

## 超级管理员

演示账号：

```text
邮箱：admin@zuopin.test
密码：Admin123
```

在注册/登录弹层使用该账号登录后，会直接进入管理平台。管理平台可查看用户、作品、评价、留言数据，给普通用户输入新密码并重置，查看用户卡片，查看、转发或删除测试作品，并删除测试留言。作品管理中会展示每件作品的评价数、好评数、差评数、好评率、好评值、差评值、当前共识、热度标签和好评程度；点击作品发布者可查看该发布者用户卡片。
